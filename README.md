# Red Square

**Open Source. Engine Powered. Chess Focused.**

♟️ Red Shapes is an open-source chess analysis platform that delivers engine-powered insights, move evaluations, and strategic recommendations. Built for accessibility and performance, it helps players analyze positions, study games, and improve their chess understanding across multiple platforms.

---

## Installation

```bash
git clone https://github.com/redshapes000/chess
```

Make sure you have downloaded git from https://git-scm.com/.
Make sure you have python from https://www.python.org/downloads/.

---

## Usage

To set up Red Shapes locally, download or clone the project files and open a terminal in the project's folder. Start a local web server by running `python -m http.server 8000`, then open your web browser and navigate to `http://localhost:8000`. Red Shapes should load immediately, allowing you to use it directly from your computer without needing any external hosting service. Make sure Python 3 is installed and added to your system PATH before running the command.

```bash
git clone https://github.com/redshapes000/chess
cd chess
python -m http.server 8000
```

## Contributing

Contributions are welcome and appreciated! Whether you're fixing bugs, improving performance, enhancing documentation, or adding new features, your help makes Red Shapes better for everyone.

### Getting Started

1. Fork the repository.
2. Clone your fork to your local machine.
3. Set up a local development environment.
4. Create a new branch for your changes.
5. Make and test your modifications.
6. Commit your changes with clear and descriptive messages.
7. Push your branch and open a Pull Request.

### Development Setup

Clone the repository and navigate to the project directory:

```bash
git clone https://github.com/redshapes000/chess
cd chess
```

Start a local web server:

```bash

```

Then open your browser and visit:

